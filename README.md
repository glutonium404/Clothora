# Clothora

A location-based fashion showcase marketplace that bridges **online discovery** with **offline buying**.

## Problem

- Buying clothes often requires visiting multiple stores
- This process is time-consuming and stressful
- Post-purchase regret when a better option is found elsewhere
- Many people are not comfortable buying clothes online — they want to touch and feel the fabric

## Solution

Clothora is a **location-based fashion showcase marketplace** where customers can:

- Discover clothing online with full details (fabric, sizes, colors, price)
- See which nearby physical stores carry each item
- Visit the store of their choice to touch, feel, and purchase
- Save time by knowing exactly where to go before leaving home

## Features

- 🏠 **Home** — Browse all clothing items with category filters and search
- 🔍 **Explore** — Search across all clothes, brands, and categories
- 👕 **Item Detail** — Full item info + list of nearby stores with distance, stock status, hours & directions
- 🤍 **Wishlist** — Save favourite items for later
- 👤 **Profile** — User profile with preferences and app info

## Tech Stack

- React 19 + Vite
- React Router v7
- Plain CSS (no external UI library)

## Getting Started

```bash
npm install
npm run dev
```
